Frequently Asked Questions

These cover the most common setup, sync, backup, and purchase questions.

How do I sync data across devices?

Sign in to the same Apple ID on each device, make sure iCloud is enabled for Repcord, and keep the app connected to the internet. You can also check the Sync Diagnostics page inside Settings.

How do I back up and restore my data?

In the Stats page, open export and choose Backup to save a JSON backup file. You can later import that file back into the app and choose whether to overwrite matching record IDs or import missing data only.

How do I restore purchases?

Open Settings, go to the purchase section, and tap Restore Purchases. If that still fails, contact support with a screenshot of the message shown in the app.

Why is a device not receiving new records?

Open Sync Diagnostics on both devices and check the latest import/export status. If export is failing, send the exact CloudKit error text shown there.

What should I include in a bug report?

Include the app version, iOS version, device model, the exact steps to reproduce the issue, and any screenshots from Sync Diagnostics or purchase alerts.
